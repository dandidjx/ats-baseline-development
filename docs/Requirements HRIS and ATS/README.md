# Requirements HRIS and ATS

This folder defines product requirements for ATS/HRIS behavior and domain entities.

## Files
- `00-Overview.md`: scope and high-level requirement summary.
- `01-Entities-Organization.md`: organization entity and related attributes.
- `02-Entities-Contact.md`: contact entity requirements.
- `03-Entities-JobOpening.md`: job opening lifecycle and required fields.
- `04-Entities-Candidate.md`: candidate entity model and constraints.
- `05-Entities-CandidateIntake.md`: intake workflow and data capture rules.
- `06-Entities-Application.md`: application entity and process coverage.
- `07-DataModel-ERD.md`: conceptual data model and ERD notes.
- `08-Architecture.md`: system architecture requirements.
- `09-UI-Mockups.md`: UI requirement references and mockup notes.
